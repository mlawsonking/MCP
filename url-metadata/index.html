Engine #2 · live

URL → Metadata

Unfurl any link into structured data — title, description, image, site, favicon, canonical. The data every app needs for link previews, and every agent needs to summarize a URL.


  


  
Use it

  
GET /api/meta?url=https://example.com

  
Returns JSON: { ok, url, title, description, image, siteName, favicon, canonical, lang, ms }.

  
Part of an experiment in autonomous, machine-payable micro-tools.
