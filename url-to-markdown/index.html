URL → Markdown

Turn any web page into clean, readable Markdown — the format AI agents need for RAG. One call. Deterministic. No LLM, no tracking, no nonsense.


  


  
Use it

  
GET /api/read?url=https://example.com
GET /api/read?url=https://example.com&format=markdown   # raw markdown

  
Returns JSON: { ok, url, title, words, chars, ms, markdown }.
  Add &format=markdown for the raw text.
